ReferralPrep

Messy referral notes in. Clean, complete clinical CSV out. One patient at a time.

The problem

Referral letters arrive messy and incomplete: a missing duration, no allergy line, medications scrawled as "aspirin OD" instead of a clear dose, and sometimes not even a patient name in a consistent place. Before any of that can become a row in a clinical database or a safe handover, someone cleans it up by hand, note after note.

ReferralPrep does that cleanup for you, and refuses to let a field go missing unnoticed.

What it does

🧾

Cleans the text

Trims spacing and expands shorthand: OD becomes once daily, NKDA becomes no known drug allergies.

⚠

Flags what is missing

Scores each referral for completeness and names any blank required field before it is saved.

📊

Builds clean CSV

Every patient becomes one tidy row you can open in Excel or Google Sheets.

🔒

Stays offline

No cloud, no account, no internet. Patient data never leaves the machine in front of you.

Why not just use Excel?

A spreadsheet stores whatever you type. It will happily accept 500 days of medication, leave the allergy field blank, and keep "Asp" in one row and "aspirin" in the next. ReferralPrep does what a cell cannot:

🚫

It guards your data

Duration must be a whole number from 1 to 100 days. A typo like 500 is refused, not stored.

💊

It standardises drugs

Asp becomes aspirin, Met becomes metformin. Teach it your own shorthands too.

✅

It checks completeness

Every row is scored and missing fields are named before the note is ever saved.

Download

After downloading, open a terminal and run the file.

Windows: open PowerShell, then type cd ~\Downloads and .\referralprep-windows-x86_64.exe. If a blue warning appears, click "More info" then "Run anyway".

Mac and Linux: open Terminal, then type cd ~/Downloads, then chmod +x followed by the file name (for example chmod +x referralprep-linux-x86_64), then run it with ./ and the file name (for example ./referralprep-linux-x86_64). On a Mac, if macOS blocks it, allow it under System Settings, Privacy and Security, Open Anyway.

The program then asks you one question at a time. Type each answer and press Enter. When you finish, open the CSV it created in Excel or Google Sheets.
